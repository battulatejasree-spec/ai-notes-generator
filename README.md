# AI Notes Generator

This project generates structured study notes using AI.

## 🚀 Live App
(https://fersu-ai-notes-generator.hf.space)

## 🛠️ Tech Stack
- LangChain
- Gemini API
- Gradio

## 📌 Features
- Generates structured notes
- Includes:
  - Definition
  - Key Concepts
  - Examples
  - Summary
- Simple and interactive UI

## 📷 How it works
Enter a topic → Click generate → Get structured notes instantly
